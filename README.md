# My_Minesweeper

This repository is a **rebuild** of the classical minesweeper game `winmine.exe` with a higher resolution to satisfy a 1080p screen, and is totally written in **C** with **Win32 APIs** only. This program runs on **Windows** and does **NOT** support cross-platform.

This repository is a project of **Visual Studio 2022**. To open this project in **Visual Studio**, just open the project file `My_Minesweeper.sln`.

Uses emojis from Baidu Tieba.

![example](./example.png)

## Features

- Larger window size for 1080p screen.
- Same UI design and operations as `winmine.exe`.
- 100% **C** codes with **Win32 APIs** only.
- Support **Windows 10** by default.

## Directory Structure

- `res/`:
  Contains resources that are used in the program.

- `src/`:
  Contains source codes of the program.

- `VSproject/`:
  Contains project files of **Visual Studio**.
